A classic **Snake game** implemented in **Python** using the `turtle` graphics library.  
The snake moves around the screen, eats food to grow, and wraps around the edges instead of dying when hitting walls.

## Features

- Classic Snake gameplay
- **Wrap-around logic**: snake appears from the opposite side when it reaches screen edges
- Grows longer when eating food
- Ends game when snake collides with itself
- Keyboard controls: W (Up), A (Left), S (Down), D (Right)
- Minimal dependencies (only Python standard library)
